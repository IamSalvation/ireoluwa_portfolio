## 📋 About

This is a professional portfolio website for **Ireoluwa Omoniyi**, an Executive & Administrative Virtual Assistant. The portfolio showcases her skills, experience, certifications, work samples, and client testimonials.

### ✨ Features

- 🎨 **Modern & Responsive Design** - Works perfectly on all devices
- ✨ **Glassmorphism Navigation** - Premium glass effect navbar
- 🖱️ **Mouse Trail Effect** - Interactive cursor glow
- 🎯 **Typing Animation** - Dynamic hero text animation
- 📊 **Animated Counters** - Numbers count up on scroll
- 🖼️ **Auto-Scrolling Carousel** - Smooth portfolio showcase
- 📜 **Scrolling Testimonials** - Marquee-style client feedback
- 📈 **Reading Progress Bar** - Shows scroll progress
- 🔝 **Back to Top Button** - Smooth scroll to top
- 🎭 **3D Image Tilt** - Interactive hero image effect
- 📱 **Mobile-Friendly** - Fully responsive hamburger menu
- 🔗 **Social Media Preview** - Open Graph tags for rich link sharing
